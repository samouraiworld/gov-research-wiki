# Intérêt collectif

## 1. Définition
dans une SCIC (Société [[Coopérative]] d'Intérêt Collectif), l'objectif n'est pas seulement de servir les intérêts des membres, mais aussi ceux d'un territoire ou d'une communauté plus large, d'où le multisociétariat. Ce principe ancre la [[Coopérative]] dans son écosystème local.
