# Excédents : équivalent coopératif du "bénéfice". Dans une coopérative, ils sont redistribués équitablement entre les membres, réinvestis dans le projet, ou affectés à des réserves impartageables. Contrairement aux dividendes, leur distribution est encadrée pour éviter une logique spéculative.
