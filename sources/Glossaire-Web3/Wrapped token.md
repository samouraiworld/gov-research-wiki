# Wrapped token [token enveloppé] : Token qui représente un autre actif sous-jacent, généralement verrouillé dans un smart contract. Exemple : WETH est une version “enveloppée” d’ETH conforme au standard ERC-20, ou des bitcoins enveloppés (WBTC, TBTC) sur Ethereum. Les wrapped tokens permettent de rendre des actifs compatibles avec la DeFi et les standards de la chaîne hôte comme ils peuvent servir à contourner les règles contraignantes du token wrappé.
