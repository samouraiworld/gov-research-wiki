# Forge logicielle [code base repo / GitHub repo] : plateforme d'hébergement de projets logiciels, correspondant à un environnement web regroupant un ensemble d'outils qui permettent le développement collaboratif et distribué de logiciels. Elle offre un portail communautaire et l'accès à un site internet, un système de gestion des versions (Git), des trackers de bugs et de fonctionnalités, une intégration continue, une gestion de documents (wiki), et d'autres services (forums, listes de discussion). Exemples : GitHub, GitLab, Framagit.
