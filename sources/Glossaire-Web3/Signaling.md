# Signaling | signalement / vote de signal

## 1. Définition
expression d'intention ou de préférence sans vote contraignant. Mécanisme utilisé dans de nombreuses DAOs pour tester le sentiment de la communauté ([[Temperature check [prise de température]]]), proche des consultations non décisionnelles dans les coopératives : dans beaucoup de DAOs, le signaling constitue la première étape d’un processus de décision : si le signal est fort, la proposition est ensuite formalisée et soumise à un vote contraignant.
