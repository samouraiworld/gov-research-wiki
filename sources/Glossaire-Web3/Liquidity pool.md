# Liquidity pool | réserve de liquidité

## 1. Définition
réserve de [[token]]s verrouillés dans un [[Smart contracts | contrats intelligents]], utilisée pour permettre des échanges automatisés dans la [[DeFi [finance décentralisée]]. Les utilisateurs déposent deux (ou plusieurs) tokens dans cette réserve ; en échange, ils reçoivent une part des frais générés par les swaps, proportionnelle à leur apport. Au lieu d’un carnet d’ordres classique, les prix sont calculés par une formule mathématique en fonction du ratio entre les [[token]]s (exemple : x·y = k).
