# Liquidity pool [réserve de liquidité]

réserve de tokens verrouillés dans un smart contract, utilisée pour permettre des échanges automatisés dans la DeFi. Les utilisateurs déposent deux (ou plusieurs) tokens dans cette réserve ; en échange, ils reçoivent une part des frais générés par les swaps, proportionnelle à leur apport. Au lieu d’un carnet d’ordres classique, les prix sont calculés par une formule mathématique en fonction du ratio entre les tokens (exemple : x·y = k).
