# Multisig wallet | portefeuille multi-signatures

## 1. Définition
portefeuille nécessitant plusieurs signatures pour valider une transaction, évitant qu'une seule personne ait le contrôle total des fonds. Configuration typique : 3 signatures sur 5 requis (3/5), ou 2/3, etc.
