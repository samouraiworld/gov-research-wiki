# Discord : Plateforme de discussion en temps réel (texte, audio, vidéo) très utilisée par les communautés Web3 pour la coordination off-chain : annonces, débats, groupes de travail, support, etc. Les salons Discord d’un projet sont souvent l’espace principal de délibération informelle et d’organisation quotidienne.
