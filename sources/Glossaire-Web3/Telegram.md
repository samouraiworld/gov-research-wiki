# Telegram

## 1. Définition
Application de messagerie chiffrée, utilisée par de nombreux projets [[Web3 [Web décentralisé]]] pour la communication rapide (annonces, support, coordination de petites équipes). Les canaux Telegram complètent souvent [[Discord]] en tant qu’outil de coordination off-chain, notamment sur mobile.
