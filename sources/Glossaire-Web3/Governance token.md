# Governance token | token de gouvernance

## 1. Définition
[Token] donnant le droit de voter sur les décisions d'une [DAO - Decentralized Autonomous Organization | Organisation Autonome Décentralisée] ou d'un protocole, généralement selon la règle d’un [Token] = une voix (ou un poids de vote proportionnel) ; Ces tokens sont souvent librement négociables d’où un risque de ploutocratie et de concentration du pouvoir dans les DAOs.
