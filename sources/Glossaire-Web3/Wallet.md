# Wallet [portefeuille]

Application (logicielle ou matérielle) permettant de générer, stocker et utiliser les clés cryptographiques qui contrôlent l'accès aux cryptomonnaies et l'interaction avec les applications décentralisées. Contrairement à un portefeuille physique, un wallet crypto ne "contient" pas directement les actifs : il stocke les clés privées qui prouvent la propriété des actifs enregistrés sur la blockchain. Peut être "hot" (connecté à internet) ou "cold" (hors ligne pour plus de sécurité). Le wallet est l'interface principale de l'utilisateur avec le Web3.
