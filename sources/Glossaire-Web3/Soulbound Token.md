# Soulbound Token [SBT - token lié à l'âme]

token non transférable, lié à une identité spécifique. Contrairement aux tokens classiques, il ne peut ni être vendu ni échangé : cette intransférabilité est inscrite dans le code. Un SBT peut représenter une réputation, une certification, une appartenance à une communauté ou un engagement particulier. Le concept est formalisé par Vitalik Buterin, E. Glen Weyl et Puja Ohlhaver (2022), dans leur proposition de Decentralized Society.
